## рҹӘӮ What is This?

It is a fully cloud-based deal automation bot. It scrapes the latest deals from Deals Websites, converts product links into affiliate links, and broadcasts each deal to a Telegram channel вҖ” all without any server or infrastructure. It runs entirely on **Google Apps Script** with a time-driven trigger.

---

## вңЁ Features

- рҹ”Қ **Auto-scrapes** new deals from DesiDime's discussed feed
- рҹ”— **Affiliate conversion** via EarnKaro API (with graceful fallback to original link)
- рҹ“ў **Telegram broadcasting** with rich Markdown-formatted messages
- рҹ“Ӣ **Google Sheets logging** вҖ” tracks every deal with timestamp, title, store, link, price
- рҹҡ« **Duplicate prevention** вҖ” skips already-posted links automatically
- рҹ§№ **Sheet auto-trimming** вҖ” keeps only the latest 1000 rows to prevent bloat
- вҳҒпёҸ **Zero infrastructure** вҖ” runs 100% on Google's cloud, no hosting required

## рҹҡҖ Setup Guide

### Prerequisites

- A Google account (for Google Sheets + Apps Script)
- A Telegram Bot token ([create one via @BotFather](https://t.me/BotFather))
- Your Telegram channel/group Chat ID
- An [EarnKaro](https://earnkaro.com) affiliate account and API token

---

### Step 1 вҖ” Copy the Script

1. Open [Google Apps Script](https://script.google.com) вҶ’ **New Project**
2. Delete the default `myFunction` code
3. Paste the contents of [`Code.gs`](Code.gs) into the editor
4. Rename the project to **Paratrooperz** (or anything you like)

---

### Step 2 вҖ” Link a Google Sheet

1. In the Apps Script editor, go to **Services** (left sidebar)  
   вҖ” OR вҖ” open a new Google Sheet and go to **Extensions вҶ’ Apps Script**
2. The script uses `SpreadsheetApp.getActiveSpreadsheet()`, so it must be bound to a sheet

> рҹ’Ў **Recommended:** Create a new Google Sheet first, then open Apps Script from **Extensions вҶ’ Apps Script** inside that sheet. This auto-binds them.

The sheet will be auto-populated with columns:

| A вҖ” Timestamp | B вҖ” Product Title | C вҖ” Store | D вҖ” Raw Link | E вҖ” Price |
|---------------|------------------|-----------|--------------|-----------|

---

### Step 3 вҖ” Add Your Credentials

At the top of `Code.gs`, replace the placeholder values:

```javascript
const TELEGRAM_TOKEN  = 'YOUR_TELEGRAM_BOT_TOKEN';   // From @BotFather
const CHAT_ID         = 'YOUR_CHAT_ID';               // e.g. -100xxxxxxxxxx
const EARNKARO_TOKEN  = 'YOUR_EARNKARO_TOKEN';        // From EarnKaro dashboard
```

---

### Step 4 вҖ” Set Up the Trigger

1. In Apps Script, click **Triggers** (clock icon in the left sidebar)
2. Click **+ Add Trigger**
3. Configure:
   - **Function:** `scrapeDesiDimeProduction`
   - **Event source:** Time-driven
   - **Type:** Minutes timer
   - **Interval:** Every **15 minutes** (or 30 minutes)
4. Save вҖ” Google will ask for permissions, grant them

That's it. The bot is now live and running on Google's cloud. вң…

## вҡҷпёҸ How It Works

1. **Fetch** вҖ” The script fetches the HTML of `desidime.com/discussed` with a browser-like User-Agent
2. **Parse** вҖ” It splits the HTML on `</article>` tags and extracts deal metadata using regex
3. **Dedup** вҖ” It reads column D of the Google Sheet to skip any links already posted
4. **Convert** вҖ” Each raw `visit.desidime.com` link is sent to EarnKaro's API to get an affiliate link. If the API fails or returns an error, the original link is used instead
5. **Broadcast** вҖ” A formatted Markdown message is sent to Telegram via the Bot API
6. **Log** вҖ” New deals are prepended to the Google Sheet for future dedup checks
7. **Trim** вҖ” If the sheet exceeds 1000 rows, old rows are deleted

## рҹҗӣ Troubleshooting

| Issue | Fix |
|---|---|
| No messages sent | Check `TELEGRAM_TOKEN` and `CHAT_ID`. Ensure the bot is an admin in the channel |
| Affiliate links not converting | Verify `EARNKARO_TOKEN` is valid. The bot will fall back to raw links |
| `Product Title Unavailable` showing | DesiDime may have changed their HTML structure. Update the regex patterns |
| Script hits execution time limit | Reduce scrape frequency or add `Utilities.sleep(500)` between iterations |
| Duplicate deals appearing | Ensure the script is bound to the correct Google Sheet |
